This chrome extension automates the scraping of profile information from LinkedIn Recruiter.

This extension is still in testing phases and will be updated periodically.


First create an airtable account.


https://airtable.com/account



Creating a Dummy Google Chrome Account:

For best results run this extension in it's own Chrome container/window
  1. Click the accounts button near the Maximize/Minimize/Close buttons in the top right.
  2. Click on the Add button at the bottom under Other Profiles.
  3. A new popup should appear, click Continue without an account.
  4. Enter any name and color scheme you want, I titled mine AutoSourcer, then click Done.
  5. Open up your new chrome account window and install the AutoSourcer extension.
  6. You will also need to disable a flag on chrome to make this more stable:

    Disable the following flag: chrome://flags/#calculate-native-win-occlusion


    Starting the Extension

  • Initial setup to get this started is pretty simple. We first need to configure our LinkedIn search to provide us with a good chunk of decent looking results. This will vary based on your positions and qualifications and supply of profiles.
  • Note: Hiding previously viewed profiles is highly recommended. This will prevent duplicate profile collection. (Only available in Recruiter Full)
  • Once you have your results where you like them, click on the first profile of the list. This should put you in a large profile scrolling mode, where you can view forward and backwards.
  • When you have the correct profile page in place, go to the extension, make sure everything still looks good, and click Start.
  • This can be run behind other windows but not minimized or Chrome will freeze the tab.

    It will also close out any other open tabs in the browser window so save your work beforehand.

    If you want to run this overnight I recommend setting up a screensaver to prevent screen burn in.

Current Features:

  • The profile view limit for recruiter lite is 2000 profiles a day. If you hit this limit you can be locked out of your account for sometimes up to 24 hours.
  • There are timers built in to randomize the time spent on a profile page. This will in turn make our usage more natural and user like and less likely to be picked up by anti automation methods.

  • Skipping viewed profiles: Will not collect any profile that has been previously viewed at any time. The default skip time is set to anywhere from 5-7 seconds.

  • This function will collect any available similar profiles that are listed on the right side of a profile. This can be in groups of 5 or 10.

    Note: There is a record limit of 1200 rows on free airtable accounts. This will quickly fill that quota as it adds 5-10 rows every profile. I recommend only using this on a paid airtable plan.

  • This function only works on full Recruiter licenses, not Recruiter Lite. It will add profiles that are open to work in a separate tab that contains more information about what the person is looking for and filters to quickly pick out appropriate profiles for likely active candidates.
  • Randomized humanlike behavior to have more realistic browsing behaviors to prevent detection from LinkedIn's anti-scraping measures.
  • Built in error reporting - Most errors will be sent to me, and hopefully fixed with future updates.
  • Option to collect similar profiles listed on the side of the profile.
  • Recruiter Full: Option to collect candidates who are open to work with their details.

Features coming in future updates:
  • Duplicate entry checking before posting to the airtable. This will allow us to save space and reduce how much noise we have in our records.
  • Profile summary including notable points on the profile, this will be an expanded field in a future release.
  • Filter for keywords extracted from the profile
  • Filter for all previously held titles
  • Optimized human like randomization
  • Notifications when reaching database limits
  • Hopefully: Ability to work in minimized mode.



If you run into a bug or think something isn't working right your best bet is to clear the cache by hitting the "Clear Cache" button at the top of the page. This will reset your settings and should clear up any problems.
